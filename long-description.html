Overview

This course will provide you with a brief introduction to creating human workflow applications using Camunda.

You will configure a simple review and approval process to gain an understanding of the steps required to develop your own applications around Human Workflow Automation.

What Will I Learn?

At the end of this course you will be able to:

  • Access a Camunda Environment
    • Create a Camunda SaaS account
    • Access Console
    • Access Modeler
    • Access Tasklist
    • Access Operate
  • Create a Cluster
  • Create a Camunda Application
    • Create a Project using the Modeler
    • Create a Process Definition using the Modeler
    • Create a Form using the Modeler
    • Configure a User Task to display a Form
  • Deploy a Camunda Application
    • Deploy a Process Definition using the Modeler
    • Deploy an updated Process Definition using the Modeler
  • Test a Process Definition
    • Start a Process Instance using the Modeler
    • Claim a Task using Tasklist
    • Populate a Form using Tasklist
    • Complete a Task using Tasklist
  • Monitor a Process Definition
    • View a Process Instance using Operate
    • Cancel a Process Instance using Operate
    • Delete a Process Instance using Operate

Target Audience

This course is targeted at the following roles:

  • Business Analyst
  • Developer
  • Administrator

Time Required: 2 Hours

Difficulty: Foundation

Applicability: This course was developed against Camunda 8.4.0 and was last modified on 12th January 2024.
