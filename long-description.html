Overview

This course provides an introduction to Robotic Process Automation (RPA) with Camunda 8.

You will learn about various components and tools within the platform and how they integrate to facilitate efficient workflow automation, laying the foundational knowledge needed for further exploration and practical application.

What You Will Learn

At the end of this course you will be able to:

  • Create and test an RPA Script
  • Create and configure an RPA Connector task
  • Configure and run an RPA Worker
  • Deploy an RPA Script and a BPMN diagram
  • Review the process instance in Operate

Target Audience

This course is targeted at the following roles:

  • Developer
  • Low Code Developer

Time Required: 30 Minutes

Difficulty: Foundation

Applicability: This course was developed against Camunda 8.7.0 and was last modified on 12th April 2025.

Prerequisite Knowledge

This course presumes that the student has the following prerequisite knowledge:

  • Basic overview of the Camunda Platform
  • Basic knowledge of BPMN
  • Account for Camunda 8 SaaS
  • Running Cluster in SaaS
  • Access Cluster via API Client Credentials

You can obtain knowledge of the above through the following Camunda Academy course:

  • Camunda - Overview
  • Camunda - Demonstration
  • Camunda - What's New
  • Camunda BPMN
  • Camunda - Getting Started with Microservice Orchestration
